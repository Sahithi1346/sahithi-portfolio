# Sahithi's Premium Portfolio 🌐

A high-performance, cinematic, and interactive personal portfolio website built with a focus on scene-based navigation and 3D interactivity. This portfolio showcases my journey as an **AIML student** and **Web Developer**.

## ✨ Key Features

- **Scene-Based Navigation**: Modular full-screen sections (Scenes) that create a high-end storytelling experience.
- **3D Project Showcase**: An interactive 3D carousel built with custom CSS and JS transforms to highlight key engineering projects.
- **Interactive Timeline**: Educational and professional experiences displayed using 3D flip-cards that reveal details and institutional photos on hover.
- **Dynamic Backgrounds**: An animated aurora background with moving glass orbs and parallax particles for a premium "glow" aesthetic.
- **Modern Dark/Light Mode**: Fully responsive theme switching with automatic favicon adaptation.
- **Glassmorphism UI**: High-end glass-style cards and navigation elements for a sleek, modern look.

## 🛠️ Technology Stack

- **Core**: Semantic HTML5 & Modern CSS3 (Vanilla)
- **Logic**: Vanilla JavaScript (ES6+)
- **Animations**: GSAP (GreenSock Animation Platform) for smooth transitions and scroll reveals.
- **Icons**: Lucide Icons (as SVGs)
- **Deployment**: Optimized for standard static hosting.

## 🚀 Local Development

Follow these steps to run the portfolio on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sahithi1346/sahithi-portfolio.git
   ```

2. **Navigate into the folder**:
   ```bash
   cd sahithi-portfolio
   ```

3. **Install Dependencies** (if using the dev server):
   ```bash
   npm install
   ```

4. **Launch the site**:
   ```bash
   npm run dev
   ```
